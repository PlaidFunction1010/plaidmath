# 112 基隆女中｜教師甄試（數學）模擬練習｜全參數化示範 v1.0

本版本將下列題目做為「可重抽數字＋自動解答」：**Q1, Q2, Q6, Q8, Q11, Q12**。  
其他題目暫以固定核對版呈現，便於你逐步補上解析或改寫為參數化。

- **Q1** 幾何面積：抽整數 `a`，答案 `a^2 + (π/2)a^2`
- **Q2** 整根二次式：重抽 `(p,q,a)`，題面同步顯示 `a`
- **Q6** 複數幾何：抽 `s,t,θ`，圓心 `(2s,0)`、半徑 `2t`，`arg(z-2s)=θ`
- **Q8** 空間距離：邊長 `L` 參數化，答案 `L·√6/3`
- **Q11** 機率：圓桌配色，蒙地卡羅近似
- **Q12** 取整和末兩位：抽 `N(1000~3000)`，以模運算快速求末兩位
