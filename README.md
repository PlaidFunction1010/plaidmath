# 112 基隆女中｜教師甄試（數學）模擬練習｜全參數化示範 v1.1.2

**為什麼本機可以、GitHub Pages 不行？**
- 多半是**腳本載入順序＋快取**：如果先掛載 App，再載入 KaTeX，初次渲染就不會啟動；或瀏覽器快取了舊的 `questions.js`。
- 這版將 KaTeX 與 App 啟動做成「**等待機制**」，並在連結後面加上 `?v=112012` 破壞快取。
- 每次動態更新（重抽數字/切換清單）後都會重新呼叫 `renderMathInElement`。

**Q2 修正**
- 題目為「兩相異**質數**」；藉由 `(p+2)(q+2)=132` 尋找質數對（例如 `p=31, q=2`），再回推出 `a=53-(p+q)`。

把整包上傳到 GitHub → Settings → Pages 啟用即可。
